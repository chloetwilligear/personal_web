Projects

Catering System

A catering supply ordering system. Written using Java 8 and OOP principles, file I/O, JUnit testing and command-line interface. Application tracks account balance, supply inventory, ordered items, order report, and user change via text file/log.

Venue Reservation System

A venue event space registration application using Java 8, JUnit for integration testing, Spring JDBC and PostgreSQL for database connectivity and management​. System allows for easy venue space search based on search criteria and data stored in databases.

TEnmo

Payment service for transferring money between friends.

An authorized account based money management system (similar to Venmo). Client-server application written using Java 8, Spring JDBC and PostgreSQL for database connectivity and management, RESTful API for web services, and Spring Boot to implement MVC pattern

CATCH

Full stack web application made for cat adoption.

Authorized users are granted access to manipulate cats, able to change cats’ availability/status, create cats, upload a photo to a specific cat profile, and delete cats. Backend written in Java 8, Spring JDBC, JUnit for integration testing and PostgreSQL for database connectivity and management. Frontend developed with Vue.js, Firebase, and Vueify. Led in UX/UI design and created logos.

MyWeather

Enter a city, zip code, or country code for current specific weather.

Frontend web application created using HTML, CSS, and vanilla Javascript. Used OpenWeatherMap for API call.
